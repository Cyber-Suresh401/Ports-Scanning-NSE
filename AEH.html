Port Scanning NSE Script

Port scanning is the process of determining which network ports are open on a particular host or network. It is an essential technique used by network administrators and security professionals to identify potential vulnerabilities and secure the network. NSE (Nmap Scripting Engine) is a powerful tool that comes with Nmap, which is a popular open-source network exploration and security auditing tool. NSE scripts are used to automate various tasks, including port scanning.

Usage

To use this script, simply run the following command:

nmap -sC -sV -p1-1000 -n -vv --script simple_port_scan demo.testfire.net -d

Output

The script will output a list of live port found on the target network, along with their IP addresses and hostname and services versions dedected.
